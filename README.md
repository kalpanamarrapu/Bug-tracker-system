# Bug Tracker System

## Tech Stack
- Java (Spring Boot)
- REST APIs
- H2 Database
- Postman for testing

## Features
- Create Bug
- View Bugs
- Delete Bug

## API Endpoints
- GET /bugs
- POST /bugs
- DELETE /bugs/{id}

## How to Run
1. Run DemoApplication.java
2. Open http://localhost:8088/bugs
3. Use Postman to test APIs
